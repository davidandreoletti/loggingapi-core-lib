libloggingapi

ABOUT
================================================================================

libloggingapi is cross platform logging API supporting:
 - loggers
 - sinks (console/file)
 - formatters

See CHANGELOG file for further details.

MOTIVATION:
================================================================================

I am writing cross-platform C++ libraries and I needed a cross platform logging 
API to log message.

INSTALLATION:
================================================================================

    cd proj
    #To build for iOS
    make build-ios
